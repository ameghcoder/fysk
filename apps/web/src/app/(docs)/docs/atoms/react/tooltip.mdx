## Usage

<CodeBlock
  language="tsx"
  code={`import { Tooltip, TooltipContent, TooltipTrigger } from "@/components/ui"
...
<Tooltip>
    <TooltipTrigger>Hover</TooltipTrigger>

    </TooltipContent>
</Tooltip>`}
/>

## Props

### TooltipContent

| Prop                      | Type                 | Default     | Description                                                      |
| ------------------------- | -------------------- | ----------- | ---------------------------------------------------------------- |
| `variant`                 | `string`             | `"default"` | Visual style: `outline`, `glass`, `glow`, `neon`.                |
| `title`                   | `React.Node`         | -           | Primary bold heading in the tooltip.                             |
| `description`             | `React.Node`         | -           | Supporting text below the title.                                 |
| `shortcut`                | `string \| string[]` | -           | Keyboard shortcut to display (e.g. `mod+K`).                     |
| `showArrow`               | `boolean`            | `true`      | Whether to display the indicator arrow.                          |
| `side`                    | `string`             | `top`       | Position of the tooltip.                                         |
| `size`                    | `string`             | `md`        | Size of the tooltip, sizes: `sm`, `md`, `lg`.                    |
| `sideOffset`              | `number`             | `4`         | Offset of the tooltip.                                           |
| `delayDuration`           | `number`             | `700`       | Delay duration in milliseconds before the tooltip is shown.      |
| `skipDelayDuration`       | `number`             | `300`       | Skip delay duration in milliseconds before the tooltip is shown. |
| `disableHoverableContent` | `boolean`            | `false`     | Disable hoverable content.                                       |

<Typography variant="blockquote">
  Accepts all standard props from Radix UI Tooltip.
