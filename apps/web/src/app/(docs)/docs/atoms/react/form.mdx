## Usage

<CodeBlock
  language="tsx"
  code={`import { Form, FormField, FormItem, FormLabel, FormControl, FormMessage } from "@/components/ui"
import { useForm } from "react-hook-form"

const form = useForm({ ... })

<Form onSubmit={form.handleSubmit(onSubmit)}>
  <FormField
    control={form.control}
    name="username"
    render={({ field }) => (
      <FormItem>
        <FormLabel>Username</FormLabel>
        <FormControl><Input {...field} /></FormControl>
        <FormMessage />
      </FormItem>
    )}
  />
  <Button type="submit">Submit</Button>
</Form>`}
/>

## Props

### Form

| Prop              | Type                                          | Default  | Description                                               |
| ----------------- | --------------------------------------------- | -------- | --------------------------------------------------------- |
| `state`           | `"idle" \| "loading" \| "success" \| "error"` | `"idle"` | The visual submission state.                              |
| `afterSubmission` | `React.Node`                                  | -        | Content/Message to show in a centered overlay on success. |

### FormMessage

| Prop    | Type                   | Default   | Description                      |
| ------- | ---------------------- | --------- | -------------------------------- |
| `state` | `"error" \| "success"` | `"error"` | The visual style of the message. |

<Typography variant="blockquote">
  Accepts all standard props from their respective Radix and HTML elements.
