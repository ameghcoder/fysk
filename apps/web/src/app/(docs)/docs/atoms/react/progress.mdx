## Props

| Prop            | Type                | Default     | Description                                           |
| --------------- | ------------------- | ----------- | ----------------------------------------------------- |
| `value`         | `number`            | `0`         | Completion percentage (0-100).                        |
| `variant`       | `string`            | `"default"` | Visual variant style.                                 |
| `size`          | `string`            | `"md"`      | Height of the bar.                                    |
| `shape`         | `string`            | `"rounded"` | Corner radius style.                                  |
| `bufferValue`   | `number`            | -           | Secondary progress layer shown behind the main value. |
| `label`         | `string`            | -           | Text label for the progress bar.                      |
| `labelPosition` | `string`            | `"outside"` | Where to place the label/value text.                  |
| `showValue`     | `boolean`           | `false`     | Shows the percentage text.                            |
| `segments`      | `ProgressSegment[]` | -           | Array of objects for a segmented progress bar.        |
| `indeterminate` | `boolean`           | `false`     | Shows an infinite loading animation.                  |
| `shimmer`       | `boolean`           | `false`     | Enables a light sweep animation across the bar.       |
| `state`         | `string`            | `"idle"`    | Syncs the color and icon to a specific status.        |

<Typography variant="blockquote">
  Accepts all standard HTML attributes.
</Typography>
