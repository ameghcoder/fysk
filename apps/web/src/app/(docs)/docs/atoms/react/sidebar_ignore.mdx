## Usage

<CodeBlock
  language="tsx"
  code={`import { SidebarProvider, Sidebar, SidebarContent, SidebarItem } from "@/components/ui"

<SidebarProvider>
  <Sidebar>
    <SidebarContent>
      <SidebarItem label="Dashboard" />
    </SidebarContent>
  </Sidebar>
  <SidebarInset>
    {/* Page Content */}
  </SidebarInset>
</SidebarProvider>`}
/>

## Props

### SidebarProvider

| Prop          | Type                                           | Default   | Description                          |
| ------------- | ---------------------------------------------- | --------- | ------------------------------------ |
| `styleMode`   | `"fixed" \| "sticky" \| "floating" \| "inset"` | `"fixed"` | The layout strategy of the sidebar.  |
| `collapsible` | `"icon" \| "none"`                             | `"icon"`  | Behavior when "closing" the sidebar. |

### SidebarItem

| Prop      | Type         | Default | Description                               |
| --------- | ------------ | ------- | ----------------------------------------- |
| `label`   | `string`     | -       | Text label for the item.                  |
| `icon`    | `React.Node` | -       | Icon component.                           |
| `active`  | `boolean`    | `false` | Whether the item is in active state.      |
| `subMenu` | `React.Node` | -       | Content to show in a collapsible submenu. |
