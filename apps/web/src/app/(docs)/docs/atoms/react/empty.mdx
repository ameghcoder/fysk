## Props

| Prop          | Type                                           | Default     | Description                        |
| ------------- | ---------------------------------------------- | ----------- | ---------------------------------- |
| `variant`     | `"default" \| "folder" \| "post" \| "details"` | `"default"` | Pre-configured icon and style.     |
| `title`       | `string`                                       | -           | The primary message.               |
| `description` | `string`                                       | -           | Supporting text content.           |
| `icon`        | `React.Node`                                   | -           | Override the default variant icon. |
| `action`      | `React.Node`                                   | -           | Slot for buttons or links.         |

<Typography variant="blockquote">
  Accepts all standard HTML attributes for the underlying element.
