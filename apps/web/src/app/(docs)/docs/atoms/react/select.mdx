## Usage

<CodeBlock
  language="tsx"
  code={`import { Select, SelectContent, SelectItem, SelectTrigger, SelectValue } from "@/components/ui"

<Select>
  <SelectTrigger>
    <SelectValue placeholder="Theme" />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value="light">Light</SelectItem>
    <SelectItem value="dark">Dark</SelectItem>
  </SelectContent>
</Select>`}
/>

## Props

### Select

| Prop              | Type                     | Default    | Description                                                 |
| ----------------- | ------------------------ | ---------- | ----------------------------------------------------------- |
| `selection`       | `"single" \| "multiple"` | `"single"` | The mode of selection.                                      |
| `value`           | `string \| string[]`     | -          | The current value(s).                                       |
| `onValueChange`   | `function`               | -          | Callback when selection changes.                            |
| `maxDisplayItems` | `number`                 | `3`        | Maximum number of tags to show in the trigger (multi-mode). |

<Typography variant="blockquote">
  Accepts all standard props from Radix UI Select.
</Typography>
