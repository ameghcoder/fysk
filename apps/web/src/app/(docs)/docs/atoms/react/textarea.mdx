## Props

| Prop        | Type         | Default     | Description                                        |
| ----------- | ------------ | ----------- | -------------------------------------------------- |
| `variant`   | `string`     | `"default"` | Visual style: `filled`, `double`, `ghost`, `deep`. |
| `autoSize`  | `boolean`    | `false`     | Dynamically adjust height to fit content.          |
| `maxHeight` | `number`     | -           | Limit for auto-resizing height in pixels.          |
| `header`    | `React.Node` | -           | Slot for elements above the text area.             |
| `footer`    | `React.Node` | -           | Slot for elements below the text area.             |
| `glow`      | `boolean`    | `false`     | Enable a subtle glowing border/shadow effect.      |

<Typography variant="blockquote">
  Accepts all standard HTML textarea attributes.
</Typography>
