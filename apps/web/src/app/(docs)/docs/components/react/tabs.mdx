## Usage

<CodeBlock
  language="tsx"
  code={`import { Tabs, TabsContent, TabsList, TabsTrigger } from "@/components/ui"

<Tabs defaultValue="account">
  <TabsList>
    <TabsTrigger value="account">Account</TabsTrigger>
    <TabsTrigger value="password">Password</TabsTrigger>
  </TabsList>
  <TabsContent value="account">Content Acc</TabsContent>
  <TabsContent value="password">Content Pass</TabsContent>
</Tabs>`}
/>

## Props

### Tabs

| Prop        | Type                                            | Default     | Description                                                          |
| ----------- | ----------------------------------------------- | ----------- | -------------------------------------------------------------------- |
| `variant`   | `"default" \| "glass" \| "underline" \| "pill"` | `"default"` | The visual style of the tab system.                                  |
| `fullWidth` | `boolean`                                       | `false`     | Whether the tabs list should take up all available horizontal space. |
| `state`     | `"idle" \| "loading" \| "success" \| "error"`   | `"idle"`    | The current interaction state.                                       |

<Typography variant="blockquote">
  Accepts all standard props from Radix UI Tabs.
</Typography>
