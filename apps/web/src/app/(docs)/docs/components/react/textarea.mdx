## Props

| Prop           | Type         | Default     | Description                                                 |
| -------------- | ------------ | ----------- | ----------------------------------------------------------- |
| `variant`      | `string`     | `"default"` | Visual style: `filled`, `double`, `ghost`, `deep`, `glass`. |
| `state`        | `string`     | `"idle"`    | Component state: `idle`, `loading`, `success`, `error`.     |
| `autoSize`     | `boolean`    | `false`     | Dynamically adjust height to fit content.                   |
| `maxHeight`    | `number`     | -           | Limit for auto-resizing height in pixels.                   |
| `header`       | `React.Node` | -           | Slot for elements above the text area.                      |
| `footer`       | `React.Node` | -           | Slot for elements below the text area.                      |
| `icon`         | `React.Node` | -           | Custom icon for the idle state.                             |
| `iconPosition` | `string`     | `"start"`   | Position of the icon: `start` or `end`.                     |

<Typography variant="blockquote">
