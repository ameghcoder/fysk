Cards support loading, success, and error states with automatic visual feedback.

## Props

### Card

| Prop          | Type                                           | Default     | Description                                    |
| ------------- | ---------------------------------------------- | ----------- | ---------------------------------------------- |
| `variant`     | `"default" \| "outline" \| "ghost" \| "glass"` | `"default"` | The visual style of the card.                  |
| `state`       | `"idle" \| "loading" \| "success" \| "error"`  | `"idle"`    | Controls the state overlay and border styling. |
| `LoaderIcon`  | `React.Node`                                   | -           | Custom loading icon override.                  |
| `SuccessIcon` | `React.Node`                                   | -           | Custom success icon override.                  |
| `ErrorIcon`   | `React.Node`                                   | -           | Custom error icon override.                    |

<Typography variant="blockquote">
  Accepts all standard HTML `div` attributes.
</Typography>
